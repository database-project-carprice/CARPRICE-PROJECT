Rental service Project

How to open our project
	- Download programs xampp then install.
	- Delete folder htdocs in directory of your xampp program.
	- Copy our folder htdocs and paste in directory of your xampp program.
	- Run xampp apache server and mySQL
	- Open this link in your browser http://localhost/phpmyadmin/ then import sql file "Car_Rental_System.sql"
	- Now you can open our project in this link http://localhost/Rentalservice
